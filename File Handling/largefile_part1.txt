Python is a high-level, object-oriented programming language used to build web apps, mobile applications, and enterprise software systems.

Python is a platform-independent language, which means code written in Python can run on any device that supports the Python Virtual Machine (JVM).
Syntax and structure are similar to C-based languages like C++ and C#.

Hello World Program
The following is a simple program that displays the message “Hello, World!” on the screen.

public class Geeks{
public static void main(String[] args) {
System.out.println("Hello World!");
}
}
Output
Hello World!
Explanation: Above program defines a class Geeks with a main method that serves as the entry point and prints "Hello World!" to the console.
